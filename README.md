## 🚀 Getting Started

This is a static HTML/CSS/JS website. Simply open `index.html` in a browser or deploy to any static hosting service.

### Deployment Options
- **GitHub Pages** — Push to a `gh-pages` branch
- **Netlify** — Connect the repo for auto-deploy
- **Vercel** — Import the project directly
